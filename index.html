My Console App

Visual Studio Code is an integrated development environment made by Microsoft for Windows, Linux and macOS.
Visual Studio Code was first announced on April 29, 2015, by Microsoft at the 2015 Build conference. A preview build was released shortly thereafter.
To set it up after installing the app, go to extensions and search for live server. Then install it. Create a folder on the desktop and open it here. Create 3 files for html, css and java script. Start Coding.

Console screen is where we check our code before displaying it on the main screen.
To print something on console, use console.log() function.
To open console screen right click on the main screen, click on inspect and then click on console.
